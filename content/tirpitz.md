## 概要

ティルピッツ（Tirpitz）は、ドイツ海軍が誇るビスマルク級戦艦の2番艦であり、姉妹艦の有名なビスマルクと同じく、第二次世界大戦において大きな歴史的役割を果たした。しかしその役割は、ビスマルクのような「派手な海戦での活躍」ではなかった。

ティルピッツはノルウェーのフィヨルドに潜み続け、実際の大型艦との砲戦はゼロ、主砲を対艦目標に向けて発砲したのは生涯でただ一度だけだった。それにもかかわらず、ティルピッツの存在はイギリス海軍を3年以上にわたって縛り続け、戦艦・空母・特殊部隊・爆撃機隊という莫大な戦力をノルウェー監視に釘付けにした。

これは軍事戦略における「**Fleet in Being（現存艦隊主義）**」——実際に戦わなくても、存在するだけで敵に多大なコストを強いることができる——という概念の、歴史上最もよく知られた実例である。

---

## 一言でいうと

> **「一度も敵の戦艦に主砲を向けることなく、3年以上イギリス海軍を縛り続けた、戦わない戦艦」**
> 

ティルピッツは「最も成功した失敗戦艦」とも、「最も恐れられた不使用兵器」とも評される。戦場での活躍ではなく、**存在そのものが武器**だった。

---

## なぜ有名なのか

ティルピッツが歴史に残る理由は「強かったから」でも「多くを沈めたから」でもない。その有名さには三つの層がある。

**① Fleet in Beingの最も純粋な実例**

ティルピッツは3年間以上、フィヨルドに潜み、イギリス海軍が常に「いつ出てくるか」を警戒し続けなければならない状況を作り出した。この「出てこないことによる脅威」という逆説が、軍事戦略史において特別な地位を与えている。チャーチルは1942年1月に統合参謀本部への書簡で「この艦の撃沈または無力化は、現在の海戦において最も重要な出来事となる」と記した。

**② 連合国の意思決定を歪め続けた**

PQ17船団事件（1942年）において、ティルピッツ自身はほとんど動いていないにもかかわらず、その「動くかもしれない」という情報だけで、イギリス海軍は34隻の護衛付き商船団に「散開せよ」という命令を下し、結果として24隻が撃沈された。これは戦艦が「姿を見せずに船団を壊滅させた」という、戦争史上比類のない事例である。

**③ 多様な戦争の断面を映す鏡**

ティルピッツを追いかけると、以下すべてが見えてくる：
- ドイツの海軍戦略とその限界（Z計画・Fleet in Being・資源制約）
- イギリスの北方戦略とノルウェー問題
- 北極船団とソ連へのレンドリース
- 特殊作戦の進化（Xクラフト潜水艦・人間魚雷）
- 空母航空戦力と陸上爆撃機の能力・限界
- 「トールボーイ爆弾」という技術革新
- 戦艦時代の終焉と航空優位の確立

## 艦名の由来

ティルピッツ（Tirpitz）という艦名は、ドイツ帝国海軍の創設者・拡張の主導者として名高い**アルフレート・フォン・ティルピッツ提督（Alfred von Tirpitz, 1849〜1930）**に因んで命名された。

ティルピッツ提督は19世紀末から20世紀初頭にかけてドイツ帝国海軍大臣を務め、当時の世界最強海軍であったイギリス海軍と対抗するため、「リスク艦隊理論（Risikotheorie）」のもとで大規模な戦艦建造計画（通称「ティルピッツ計画」）を推進した人物である。イギリスとの海軍軍拡競争を主導したという意味で、ティルピッツ提督の名前は「英独海軍の対立」そのものを象徴していた。

皮肉なことに、その名を冠した戦艦は、理論上「艦隊決戦でイギリスに対抗する」ために建造されながら、実際には一度も敵の主力艦隊と戦うことなくフィヨルドに潜み続けた。提督の攻撃的な戦略思想とは正反対の存在になった点も、歴史的な逆説として興味深い。

## 建造された時代背景

### ドイツ海軍の再建

ティルピッツが建造された背景を理解するには、第一次世界大戦後のドイツ海軍がどのような状況に置かれていたかを知る必要がある。

1919年のヴェルサイユ条約によって、ドイツ海軍はほぼ壊滅的な制限を受けた。ドイツ艦隊の主力艦はスコットランドのスカパ・フローで自沈させられ（1919年6月）、残された海軍には新型艦建造の道もほぼ閉ざされた。

1933年にヒトラーが政権を掌握すると、ドイツは段階的に条約の束縛から脱しはじめる。1935年には英独海軍協定が締結され、イギリスはドイツがイギリス海軍の35%規模の海軍を保有することを事実上容認した。これはヴェルサイユ体制の部分的な崩壊であり、ドイツ海軍再建の公式なスタートラインとなった。

### ヴェルサイユ条約の制約

| 条項 | 内容 | ドイツへの影響 |
| --- | --- | --- |
| 戦艦保有制限 | 10,000トン以下の艦のみ | 新型主力艦の建造不可 |
| 潜水艦禁止 | Uボートの保有・建造禁止 | 主力的な海上戦力の喪失 |
| 艦艇数制限 | 戦艦6隻・巡洋艦6隻など | 質・量ともに大幅制限 |
| 人員制限 | 海軍将兵15,000名上限 | 専門人材の蓄積困難 |

しかしドイツはこの制約を「額面通り」には受け入れなかった。ポケット戦艦（装甲艦）の建造で条約の抜け穴を利用し、秘密のUボート乗員訓練を行い、1935年の英独海軍協定後は公然と大型艦建造を開始した。ビスマルクとティルピッツはその産物であり、当初35,000トンと申告しながら実際には42,000〜53,000トンに達する艦だった。

### ドイツ海軍戦略

ドイツ海軍は「どのようにイギリス海軍と戦うか」について、内部で深刻な論争を抱えていた。

**レーダー派（水上艦重視）**の戦略は、大型戦艦を核とした艦隊で大西洋の通商路を攻撃することにあった。これが**Z計画**（1939年策定）として体系化され、1948年までに戦艦10隻・空母4隻・重巡洋艦5隻などを整備する構想だった。ビスマルク・ティルピッツはその先行建造艦だった。

```
【Z計画の構想と現実】

構想（1939年）
→ 1948年までに戦艦10隻・空母4隻・重巡洋艦5隻を整備
→ イギリス海軍に対抗できる「バランスのとれた艦隊」を目指す

現実（1939年9月）
→ ポーランド侵攻で開戦。計画は完成前に中断
→ 保有可能な大型艦はビスマルク・ティルピッツ・シャルンホルスト・グナイゼナウ・プリンツ・オイゲンなど、わずか
→ 「艦隊は面目のためにしか戦えない」とレーダーは自嘲
```

**デーニッツ派（潜水艦重視）**は、島国イギリスの弱点であるシーレーンをUボートで攻撃する戦略を主張した。この対立は戦争中を通じて続き、ビスマルクの喪失後はさらに潜水艦重視へシフトした。

### イギリスとの対立

ドイツ帝国時代のティルピッツ計画以来、英独海軍の対立は深い歴史的根拠を持っていた。イギリスは島国であり、海上補給路が国家の生命線だった。ドイツの大型戦艦の出現は、イギリスにとって単なる軍事的脅威を超えた「悪夢の再現」を意味した。

ビスマルクが1941年5月にHMSフッドを撃沈したとき、イギリス国民が受けた衝撃は計り知れなかった。そのビスマルクと同型の艦がノルウェーのフィヨルドに潜んでいる——これがティルピッツに対するイギリスの過剰ともいえる警戒態勢の心理的背景だった。

ヴィルヘルムスハーフェン海軍工廠で起工。

【1939年4月1日】
進水式。アドルフ・ヒトラー総統臨席のもと、

【1940年〜1941年初頭】
慣熟訓練・装備調整。バルト海での訓練期間。
1941年2月25日、正式就役。

【1941年5月】
姉妹艦ビスマルクが「ライン演習作戦」で出撃するが、
ティルピッツは訓練未完了のため参加せず。
ビスマルクは5月27日、北大西洋で沈没。
→ ティルピッツはドイツ海軍唯一の超弩級戦艦となる。

【1942年1月中旬】
ティルピッツがノルウェーのトロンハイム・フィヨルドへ移動。

【1942年3月6〜9日：コンボイPQ12を追撃しかけるが帰投】
ティルピッツ・提督スキームら艦隊がPQ12船団を追撃するため出撃。
しかし空母HMS Victoriousからのソードフィッシュ雷撃機12機に攻撃され、
命中なしも危険を察知。ヒトラーの命令で帰投。
→ 初の実戦的行動だったが、積極作戦への移行は阻まれる。

【1942年7月4日：PQ17船団の悲劇】
ティルピッツはアルテンフィヨルドを出撃したが、
間もなく帰投命令を受け実際には交戦しなかった。
しかし「ティルピッツが出撃した」との情報だけで、
英海軍第一海軍卿ダドリー・パウンドが「船団は散開せよ」と命令。
34隻のうち24隻が沈没、153名の船員が死亡。
→ ティルピッツが「戦わずして」壊滅させた唯一の「戦闘」。

【1942年10月〜11月】
一時トロンハイムに帰投し修理・補給。

カーフィヨルド（アルタフィヨルド支流）へ移動。

1942年12月のバレンツ海海戦でドイツ重巡洋艦がイギリス護衛艦隊に撃退された事件が
ヒトラーの逆鱗に触れ、水上艦艇による積極作戦は事実上禁じられる。
レーダー海軍総司令官は解任され、デーニッツが後任に。

ティルピッツとシャルンホルスト、駆逐艦9隻がスピッツベルゲン島の
連合軍気象観測所を攻撃。ティルピッツの主砲（38cm砲）が
対地支援射撃で合計52発を発射。
これが生涯唯一の「対地・対艦目標への主砲発射」。
→ ノルウェーの軍事的係留地の守備隊は9名死亡・31名捕虜。
→ この「戦闘」のために乗員には400個のアイアンクロスが授与された。

イギリスXクラフト超小型潜水艦X6・X7がカーフィヨルドに侵入成功。
艦底に4トン爆薬を設置・起爆。
ティルピッツは18m規模の船底亀裂を受け、
機関・電気系統・測距装置に深刻な損傷。
修理に6ヶ月以上を要した。
→ X6艦長キャメロン中尉、X7艦長プレース中尉が捕虜となりVCを授与。

空母HMS Victorious・HMS Furious他5隻の空母艦載機

2波にわたる攻撃で爆弾命中多数（14発以上命中との記録）。
乗員128名死亡・320名以上負傷という大きな人的損害を受けたが、
ティルピッツ本体は致命的損傷に至らず。
修理に約3ヶ月。

空母艦載機44機（バラクーダ・護衛機40機）が攻撃するも、
ドイツ軍の煙幕に阻まれ命中弾なし。

【1944年8月22日〜29日：Operation Goodwood（グッドウッド作戦）】
空母艦載機による繰り返しの攻撃（4回の打撃）。
8月24日に爆弾2発命中するも致命傷には至らず。

【1944年9月15日：Operation Paravane（パラベーン作戦）】
RAFランカスター爆撃機28機（第9飛行隊・第617飛行隊）が
5トン「トールボーイ爆弾」1発命中。
船体に重大な損傷を負い、戦闘艦としての運用能力を喪失。
ドイツ海軍はティルピッツを戦闘任務から外し、
ノルウェー沿岸防衛の「浮き砲台」として活用することを決定。

【1944年10月29日：Operation Obviate（オブビエート作戦）】
ランカスター38機が攻撃するも、悪天候で目標が雲に隠れ、
至近弾1発のみ。致命打に至らず。

【1944年11月12日：Operation Catechism（カテキズム作戦）最期】
午前3時すぎ、RAF第617飛行隊（ダムバスターズ）と第9飛行隊の
ランカスター32機が北スコットランドのロッシーマスを出撃。
午前9時38分、ティルピッツが北方約190海里（北極圏内）トロムソ沖に錨泊中に接近。
ティルピッツが煙幕を張る間もなく、爆弾が次々と投下される。
トールボーイ2〜3発が直撃、数発が至近弾となり、
艦内で大爆発が発生。ティルピッツは急速に傾斜。
約11分後に完全に転覆・着底。
乗員1,900名のうち約1,000名が死亡または行方不明。
連合国側の損害：ランカスター1機。
→ この日をもって、ドイツ海軍最後の超弩級戦艦が消えた。
```

## ドイツ海軍戦略とティルピッツ

### Fleet in Beingとは何か

「Fleet in Being（フリート・イン・ビーイング）」とは、17世紀末のイギリス海軍提督アーサー・ハーバート（トリントン伯）が提唱した海軍戦略概念で、日本語では**「現存艦隊主義」**と訳される。

その本質は「**実際に戦わなくても、存在するだけで敵の行動を制約できる艦隊**」という考え方である。

| 通常の艦隊運用 | Fleet in Being |
| --- | --- |
| 積極的に出撃・戦闘 | フィヨルドに潜み待機 |
| 敵を撃破することで脅威を除去 | 潜在的脅威として存在し続けることで敵を拘束 |
| 損害を受けるリスクあり | 消耗しないので脅威が持続 |
| 作戦効果は即時的 | 作戦効果は長期的・間接的 |

現代でいえば、空母打撃群の存在が他国の行動を制約するのと同じ論理である。北朝鮮の核兵器も「使われないが存在する」ことで抑止力として機能するという意味では同じ構造を持つ。

### なぜティルピッツは脅威だったのか

ティルピッツが「脅威」であったことには複数の層がある。

**① 物理的能力**
38cm砲8門を持つ超弩級戦艦は、当時の世界最強水準の兵装を持っていた。北極船団の護衛艦隊（駆逐艦・巡洋艦主体）では対抗不可能であり、もし出撃すれば護衛戦力を圧倒する能力があった。

**② 地理的優位**
ノルウェーのフィヨルドは天然の要塞だった。山に囲まれ、対潜ネット・機雷・対空砲台に守られた碇泊地は、通常の攻撃手段では到達が困難だった。さらに夏は極夜がなく夜間攻撃が難しく、冬は嵐と視界不良で空爆が制限された。

**③ 心理的プレッシャー**
ビスマルクがHMSフッドを数分で撃沈した記憶が、イギリス海軍将兵の心に刻まれていた。姉妹艦のティルピッツが北方に潜んでいるという事実は、論理的な脅威評価を超えた「恐怖」として機能した。

> 「ティルピッツの撃沈または無力化は、現在の海戦において最も重要な出来事となる。（略）全ての他の任務はこの目的のために後回しにするべきである。」
— ウィンストン・チャーチル、1942年1月、統合参謀本部への書簡
> 

これほど強い言葉で「一隻の艦の破壊」を命じた首相の発言は異例であり、ティルピッツがいかに特別な脅威と見なされていたかを示している。

**④ 情報の不確かさ**
イギリスの情報機関（OIC）は常にティルピッツの動向を監視していたが、完全な情報を持つことはできなかった。特にエニグマ解読が一時停止した期間や、情報が錯綜した局面では、最悪のケース（ティルピッツ出撃）を想定して動かざるを得なかった。PQ17事件はその典型例である。

### なぜほとんど出撃しなかったのか

ティルピッツが積極的に出撃しなかった理由は複合的である。

**① ヒトラーの大型艦保護命令**

ビスマルク喪失（1941年5月）はヒトラーに深刻なショックを与えた。ドイツ海軍の威信の象徴だった姉妹艦の喪失は、彼を大型艦の積極的運用に反対する方向に転換させた。1942年12月のバレンツ海海戦後、ヒトラーは水上艦艇部隊の積極的運用を事実上禁止した。

**② 燃料不足**

第三帝国の重油事情は戦争が進むにつれて悪化した。戦艦1隻の出撃に必要な燃料は莫大であり、これを確保できない状況が長期化した。1943年以降、ティルピッツはカーフィヨルドを出るためだけの燃料供給すら困難になっていた。

**③ 航空支援の欠如**

ドイツ海軍は空母を持たなかった（グラーフ・ツェッペリンは未完成）。空母や陸上航空機の援護なしに出撃すれば、イギリスの空母艦載機の格好の標的になる。ビスマルクを沈めたのがソードフィッシュの魚雷だったことをドイツ海軍は知っており、航空支援なしの出撃は「死に行くこと」と同義だった。

**④ 護衛できなかったコスト計算**

戦艦1隻の安全な出撃には多数の護衛駆逐艦が必要だが、ドイツ海軍にはその余裕がなかった。Uボート作戦や沿岸防衛にも駆逐艦が必要であり、ティルピッツの出撃支援に割く余力がなかった。

**⑤ 「存在が価値を持つ」という逆算**

レーダー、後にはデーニッツも理解していたことは「ティルピッツが生きている限り、イギリスはその対処に資源を割かなければならない」という事実だった。つまり出撃して消耗するよりも、フィヨルドで生き続ける方が戦略的に価値があるという逆算が成立していた。

## 北極船団との関係

### ソ連支援の重要性

北極船団（Arctic Convoys）を理解するには、なぜ連合国がそこまで危険な航路でソ連に物資を送り続けたかを理解する必要がある。

1941年6月のバルバロッサ作戦（独ソ戦開始）から、ソ連は西部戦線で激戦を繰り広げていた。ソ連の戦時生産能力は甚大な被害を受け、特に1941〜42年にかけての生産力の低下は深刻だった。アメリカの武器貸与法（レンドリース）によるソ連への援助は、大きく分けて3つのルートで送られた。

| 補給ルート | 概要 | 特徴 |
| --- | --- | --- |
| 北極ルート（北極船団） | スコットランド〜ノルウェー海〜ムルマンスク・アルハンゲリスク | 最短距離だが最も危険。ドイツ空軍・海軍・潜水艦の攻撃圏内 |
| イラン・ルート | 中東経由でイランからソ連へ | 長距離だが比較的安全 |
| 太平洋ルート | アメリカ西海岸→日本に近い北太平洋経由 | 日本が中立を保った時期に有効 |

北極ルートは距離が最短で物資を最速で届けられる反面、ノルウェー沿岸のドイツ基地から航空・海上・潜水艦攻撃を受ける最も危険なルートだった。

**レンドリースでソ連が受け取ったもの（主な品目）：**

| 品目 | 数量 | 意義 |
| --- | --- | --- |
| 航空機 | 約14,000機 | ソ連空軍の再建に貢献 |
| 戦車・装甲車 | 約13,000両 | スターリングラード・クルスク戦での機甲戦力を補完 |
| 電話線 | 約161,000km分 | 通信インフラの確保 |
| 食料 | 約450万トン | 前線兵士・市民の糧食 |
| 自動車・トラック | 約50万台 | 機動力・兵站の革命的改善 |

ソ連の軍事指導者ゲオルギー・ジューコフは後年、「アメリカの食料と輸送機材がなければ、我々はおそらく1943年に戦争に負けていた」と述べたとされる。北極船団は「ドイツとの戦争でソ連が戦い続けられるかどうか」を左右するインフラだった。

### PQ17船団事件

PQ17事件は、ティルピッツの Fleet in Being が生んだ最大の惨劇であり、同時に「戦艦が一発も撃たずに船団を壊滅させた」という軍事史上唯一の事例として歴史に残る。

```
【1942年6月27日〜7月10日：PQ17の経緯】

【6月27日】
PQ17船団（商船34隻、護衛艦多数）がアイスランドのフヴァルフィヨルドを出港。
護衛は巡洋艦4隻、駆逐艦3隻、その他多数の補助艦艇。
さらに遠距離支援として戦艦HMS Duke of York・アメリカ戦艦USS Washington・空母HMS Victoriousなどが後方に待機。
↓
【7月1日〜2日】
ドイツ軍が船団を発見。Uボート・航空機による攻撃が始まる。
この時点では護衛艦が撃退し、被害は軽微。
↓
【7月4日 朝】
船団はバレンツ海に入る。
ロンドンのOIC（海軍作戦情報センター）で
「ティルピッツが出撃した可能性がある」という断片的情報が錯綜。
↓
【7月4日 21:11】
英海軍第一海軍卿ダドリー・パウンド提督が
他の参謀の反対意見を押し切り、自らの判断で命令を発令。
「巡洋艦部隊、高速で西方に撤退せよ」
↓
【7月4日 21:23】
「水上艦の脅威のため、船団は分散してロシア各港へ向かえ」
↓
【7月4日 21:36】
「船団は散開せよ（Convoy is to scatter）」

この「散開命令」は、実際にティルピッツが出撃していたかどうかの
確認が取れていないまま発令された。
OICは「ティルピッツはまだフィヨルドにいると思われる」と意見していたが
パウンドはこれを無視した。
↓
【7月5日〜10日】
護衛を失った商船は、ドイツ潜水艦と航空機の格好の標的となった。
各自バラバラに逃げる商船を、Uボートと航空機が組織的に狩る。
↓
【最終結果】
・34隻の商船のうち24隻が沈没（商船の71%）
・153名の船員が死亡
・積み荷：戦車430両・航空機210機・自動車3,350台・一般貨物10万トン以上を喪失
・これはイギリス史上、一つの護送船団への最大の損害

そして——
ティルピッツは実際にはアルテンフィヨルドに碇泊したままだった。
出撃はわずかな距離にとどまり、間もなく帰投命令を受けた。
```

**PQ17事件の歴史的意義：**

この事件はただの海軍の失敗以上の意味を持つ。ティルピッツという「可能性の脅威」が、何千トンもの物資とソ連への補給を消し去った。ティルピッツは一発も撃たずに、PQ17船団を壊滅させた。これがFleet in Beingの純粋な成功事例として歴史書に刻まれる理由である。

また、ダドリー・パウンド提督の判断も論争の的となった。彼は当時、脳腫瘍を患っていたことが後に判明し、判断力の低下があったとする説もある。いずれにせよ、正確な情報が得られないまま最悪の想定に基づいて行動した結果が、自ら招いた惨劇だった。

### 北極海における心理的影響

PQ17事件後、連合国はノルウェー海を通る北極船団の運航を一時停止した（1942年8月〜11月）。この間、ソ連への物資輸送は大きく制限された。

ティルピッツが存在するだけで、年間を通じた北極補給を断続させる効果があった。1943年以降も、護衛に戦艦クラスの艦艇が必要という状況が続き、イギリス本国艦隊の戦力の一部は常にティルピッツへの対応に割かれ続けた。

チャーチルとスターリンの関係においても、この補給問題は緊張の源だった。スターリンは北極船団の停止に強く抗議し、チャーチルはその説明に苦労した。「なぜ同盟国が同盟国への補給を停止するのか」——その答えの核心には常にティルピッツの存在があった。

## イギリス軍特殊作戦

### Operation Source（ソース作戦）1943年9月22日

**背景：**
Operation Tunsten以前に、イギリスは特殊な手段でしかティルピッツに到達できないことを認識していた。カーフィヨルドは北極圏の奥深くにあり、対潜ネット・機雷・護衛艦・対空砲台に守られ、通常の艦艇・航空機による攻撃は「距離と防御」の両面で困難だった。

そこで開発されたのが**Xクラフト（X-craft）超小型潜水艦**だった。全長約16m、乗員4名、側面に設置した爆薬（4トン）を目標物の下に置いてくるという機能を持つ特殊兵器だった。

```
【ソース作戦の経緯】

【9月11日】
スコットランドを出発。
通常潜水艦6隻がそれぞれXクラフトを曳航。
目標：X5・X6・X7→ティルピッツ
   X8→リュッツォウ（ポケット戦艦）
   X9・X10→シャルンホルスト

【9月11〜20日：北海横断中の損失】
・X9：曳航索が切れ、乗員全員死亡
・X8：バラストタンク浸水→爆薬の一方が早期爆発→修理不能で自沈
→ ティルピッツへの攻撃は実質X5・X6・X7の3隻が担うことに

【9月22日 01:00頃】
3隻のXクラフトが曳航索を切り離し、独力でカーフィヨルドへ侵入開始。

【9月22日 朝】
・X6（キャメロン中尉）：防潜ネットをくぐり抜け、ティルピッツ艦底付近に到達。
  しかし最後のネットに絡まり艦体を損傷。浮上後、艦を放棄・自沈。
  爆薬を設置済み。キャメロン中尉ら乗員はドイツ兵に捕虜として救出される。

・X7（プレース中尉）：ネットを突破し爆薬を設置。
  ネットに絡まって脱出不能。爆発を艦内で経験。
  プレース中尉は脱出に成功し浮上。乗員は捕虜に。

・X5：消息不明。艦体と乗員はカーフィヨルド内で発見されず。
  ドイツ軍艦艇に発見され撃沈されたとも言われる。

【9月22日 08:12】
4トン爆薬が起爆。
ティルピッツが大きく揺れ、船底に約18メートルの亀裂が生じる。
機関室・タービン室・舵装置に深刻な損傷。
照準装置・射撃管制装置の多くが使用不能に。

【結果】
・ティルピッツは約6ヶ月間、戦闘不能状態に。
・キャメロン中尉・プレース中尉はVCを授与。
・ドイツ軍はほぼ全乗員を救助したが、2名が戦死（機関室内）。
```

この作戦はのちに映画化・書籍化され、特殊部隊史の代表的事例として語り継がれている。カーフィヨルドのような難攻不落の場所に16mの小型潜水艦が侵入できたこと自体が驚異的だった。

### Operation Tungsten（タングステン作戦）1944年4月3日

**背景：**
Operation Sourceでの損傷が修復されつつあるという情報をイギリスは掴んでいた。1944年4月初旬に公試運転が予定されているという諜報もあり、「動き始める前に撃つ」機会として、大規模な空母艦載機攻撃が計画された。

```
【タングステン作戦の構成】

攻撃参加艦艇：
・空母HMS Victorious、HMS Furious
・護衛空母HMS Emperor、HMS Pursuer、HMS Searcher

攻撃機：
・フェアリー・バラクーダ急降下爆撃機 42機（第1波21機・第2波21機）
・護衛戦闘機（コルセア・ヘルキャット・ワイルドキャット等）約40機以上

【4月3日 05:30頃】
第1波バラクーダ21機がカーフィヨルドへ向かう。
驚異的な奇襲成功。煙幕を張る間もなく攻撃を受ける。
命中弾：少なくとも7発以上（1,600ポンド爆弾・500ポンド爆弾）

【4月3日 06:35頃】
第2波バラクーダ21機が追撃。
さらなる命中弾を与える。

【結果】
・ティルピッツ乗員：死者128名、負傷者320名以上
・しかしティルピッツ本体は沈まず。装甲が主要部分を守りきった。
・修理期間：約3ヶ月
・連合国機の損失：5機
```

**タングステン作戦の逆説：**
艦載機の攻撃で100名以上の乗員を死亡させながら、戦艦を沈めることはできなかった。これは「空母艦載機の爆弾では、戦艦の装甲を抜くことができない」という当時の技術的限界を示した。ティルピッツの装甲（最大320mm）は艦載機の爆弾の貫通能力を上回っていた。これが後にRAFの爆撃機部隊に任務が移管される理由となる。

### Operation Mascot（マスコット作戦）1944年7月17日

**背景：**
タングステン作戦の損傷が修復されたという情報に基づき、再攻撃が計画された。しかし今回はドイツ側が完全な準備を整えていた。

```
【マスコット作戦の経緯】

攻撃機：バラクーダ44機 + 護衛戦闘機40機
出発：空母HMS Formidable・HMS Indefatigable・HMS Furious

しかし攻撃機がフィヨルドに到達する直前、
ドイツ軍の煙幕発生システムが作動。
カーフィヨルドはあっという間に濃密な人工霧に覆われた。

バラクーダは視界を失い、ティルピッツの位置が確認できない。
命中弾ゼロ。攻撃機3機を失い撤退。
```

マスコット作戦は「煙幕による防空戦術が、航空攻撃を完全に無力化できる」ことを証明した事例として興味深い。デジタル・レーダーがない時代、視界を奪うという原始的な戦術が、最新鋭の空母艦載機を完全に封じた。

### Operation Goodwood（グッドウッド作戦）1944年8月22〜29日

**背景：**
マスコット作戦の失敗を受け、イギリス海軍は「繰り返しの攻撃で防衛側を疲弊させる」戦略を採用した。8月22日から29日にかけて4回の打撃（Goodwood I〜IV）が実施された。

```
【グッドウッド作戦の結果】

Goodwood I（8月22日午前）：主要な被害なし
Goodwood II（8月22日夕方）：主要な被害なし
Goodwood III（8月24日）：爆弾2発命中（1発1,600ポンド・1発500ポンド）。しかし致命傷なし
Goodwood IV（8月29日）：命中弾なし

連合国側の損失：航空機17機（全原因）、護衛艦1隻が潜水艦に撃沈、護衛空母1隻損傷
ドイツ側損失：航空機12機、艦船7隻損傷
```

グッドウッド作戦の失敗は、「艦載機と通常爆弾では、防御された戦艦を沈めることができない」という技術的限界の結論を出した。この時点でイギリス海軍は率直に認めた——ティルピッツを沈める仕事はRAF（王立空軍）のランカスター爆撃機と「トールボーイ爆弾」に委ねるべきだ、と。

### Operation Catechism（カテキズム作戦）1944年11月12日

**背景と経緯：**
9月15日のOperation Paravaneでトールボーイ爆弾1発が命中し、ティルピッツに深刻な損傷を与えた。ドイツ海軍はティルピッツを「浮き砲台」として活用するためトロムソ沖に移動させ、修理の完全な見込みもないまま放置していた。

10月29日のOperation Obviateはほぼ失敗に終わったが、11月12日が最後の機会となった。

```
【カテキズム作戦】

参加部隊：
・第617飛行隊（ダムバスターズ）：ランカスター18機
・第9飛行隊：ランカスター13機（記録によって差異あり、計30〜32機）
出発地：スコットランド、ロッシーマス

爆装：各機1発、12,000ポンド（5.4トン）「トールボーイ爆弾」

特別改造：
・中部上部銃座撤去（重量軽量化）
・爆弾倉扉を膨らんだ形状に改造（トールボーイが入るよう）

ルート：
・ノルウェー海を越え、スウェーデン上空を通過（スウェーデンの中立侵害）
・トロムソ沖約190海里（北極圏内）まで進出
・往復距離：約2,200マイル

**トールボーイ爆弾とは：**

## 関連する歴史事件

### 第二次世界大戦全体との関係

ティルピッツはノルウェーという地理的に北方の場所に位置しながら、欧州戦争全体の戦略に影響を与え続けた。

- **大西洋の戦い（Battle of the Atlantic）への影響：** ティルピッツが北方に居続けることで、イギリス本国艦隊は大西洋の対潜作戦や他の任務に充分なリソースを割けなかった。
- **ノルマンディー作戦（1944年6月）への影響：** ティルピッツが健在である限り、イギリス海軍はノルウェー方面の警戒を緩めることができなかった。ティルピッツの最終的な無力化（1944年9月）は、連合国側がノルマンディー以降の作戦に資源を集中する環境整備に貢献した。

### バレンツ海海戦（1942年12月31日）

PQ17事件と並んで、ドイツ海軍水上艦の消極性を象徴する事件。重巡洋艦ヒッパー・リュッツォウと駆逐艦隊がJW51B船団を攻撃しようとしたが、イギリス護衛駆逐艦隊の激しい抵抗により損害を受けて撤退。

ヒトラーはこの「失態」に激怒し、「水上艦は全部スクラップにしろ」と叫んだとされる。この発言でレーダー海軍総司令官が辞任し、デーニッツが後任に就いた。この人事変更はドイツ海軍戦略における水上艦の地位低下を決定的にした。

### シャルンホルスト撃沈（1943年12月26日：北岬海戦）

1943年12月、シャルンホルストがJW55B船団を攻撃するために出撃したところを、HMS Duke of York（戦艦）率いるイギリス艦隊に捕捉・撃沈された（北岬海戦）。

この撃沈によりティルピッツは、就役中のドイツ海軍唯一の超弩級戦艦となった。同時に、シャルンホルストの敗北は「護衛なしの単艦行動の危険性」を再確認させ、ティルピッツの出撃をさらに困難にした。

### Operation Chariot（セント・ナゼール強襲、1942年3月）

直接的にはティルピッツと無関係に見えるが、実は密接な関係がある。ティルピッツが大西洋に進出した場合、修理できる唯一の大型ドックがフランスのサン・ナゼールにあった（ノルマンディードック）。イギリス軍はこのドックを破壊するためにHMS カンベルタウン（旧式駆逐艦）を突撃させ、爆破した。

これによりティルピッツは「大西洋に出ても修理できる港がない」状況になった。この一因もティルピッツがノルウェーに縛り付けられた理由の一つとなった。

## 歴史的評価

### 「最も成功した失敗戦艦」という逆説

ティルピッツはしばしば「最も成功した失敗戦艦」と評される。これは矛盾しているように聞こえるが、二つの文脈から理解できる。

**「失敗」の側面：**
- 主砲を敵の戦闘艦艇に向けて発砲したことは一度もない
- 沈めた商船数は、単艦としてはほぼゼロ
- 建造コストと維持コストは莫大で、Uボート数十隻に相当する
- 最終的には沈められた（艦としての「負け」）

**「成功」の側面：**
- 3年以上、イギリス本国艦隊の戦艦・空母を北方に拘束し続けた
- PQ17船団壊滅を（一発も撃たずに）達成した
- 北極船団の運航を繰り返し遅延・停止させた
- イギリスの特殊部隊・空母艦載機・爆撃機隊という莫大な資源を消耗させた
- ノルウェー占領の正当化と北方防衛の象徴となった

実際にどれほどの連合国資源が拘束されたか：
- 常時2隻以上の戦艦・1隻以上の空母を北方に待機させる必要があった
- 繰り返された攻撃作戦（合計9回以上の大規模攻撃）で航空機・人員を消耗
- 北極圏での偵察飛行・情報収集に多大な資源が投入された

学術論文「Fleet-in-Being: Tirpitz and the Battle for the Arctic Convoys」（Saber and Scroll誌）は「第二次世界大戦においてティルピッツほど、実際の戦闘成果に対して不釣り合いに多くの連合国資源を拘束した艦は他にない」と評価している。

### 戦艦時代の終焉を告げた事件としての意義

ティルピッツの沈没は、単なる一隻の戦艦の喪失ではなく、以下の意義を持つ。

**「陸上爆撃機が戦艦を沈めた」という事実：**
それまでの常識では「戦艦を沈めるには、より強力な戦艦か空母艦載機が必要」とされていた。しかしティルピッツを最終的に沈めたのは、陸上基地から飛来したRAFのランカスター爆撃機だった。これは「制空権を持つ側が制海権を持つ」という航空優位の原則の最終確認だった。

ビスマルクを「旧式複葉機が沈めた」とすれば、ティルピッツを「5.4トン爆弾を持つ陸上爆撃機が沈めた」という事実は、戦艦の時代が完全に終わったことを宣告している。

## 論争点・誤解されやすい点

### 論争1：ティルピッツの Fleet in Beingは「戦略的意図」だったのか？

| 立場 | 主張 | 根拠 |
| --- | --- | --- |
| 意図された戦略説 | ティルピッツをフィヨルドに置くことでイギリスを拘束する効果は、ドイツ海軍が意図したものだった | レーダーは水上艦の「存在の価値」を認識しており、温存は合理的選択だった |
| 能力・制約の結果説 | 実際には燃料不足・ヒトラーの禁止令・航空支援の欠如で「出たくても出られなかった」だけ | 燃料供給記録・出撃命令の経緯・ヒトラーとレーダーの対立がこれを示す |
| 現代の主流評価 | 両方が複合的に作用した。「意図していた効果」と「制約による結果」の区別は難しく、結果として Fleet in Being になった | 複数の歴史的資料が双方の側面を示している |

### 論争2：PQ17の「散開命令」は正しかったのか？

| 立場 | 主張 |
| --- | --- |
| パウンド擁護説 | 当時得られる情報では、ティルピッツ出撃を最悪のケースとして想定するのは合理的。結果論で批判するのは不公平 |
| パウンド批判説 | OICはティルピッツがまだフィヨルドにいると判断していた。この意見を無視したのは判断ミス。さらに脳腫瘍による判断力の低下が影響していた可能性がある |

### 論争3：ティルピッツへの投資はドイツにとって正しかったのか？

| 視点 | 内容 |
| --- | --- |
| 否定的評価 | ティルピッツ建造・維持に投じた資源でUボートを数十隻建造・運用できた。Uボートの方が大西洋戦争で遥かに大きな効果を発揮していた |
| 肯定的評価 | Fleet in Beingとしての拘束効果は、Uボートでは代替できない独自の戦略価値があった。「潜在的脅威」という心理的効果は水上大型艦にしか生み出せない |
| 時代的必然論 | Z計画が始まった時点では「大型艦が海軍力の象徴」という時代観が支配的であり、当時の判断基準では不合理とは言えない |

### 誤解①：「ティルピッツはPQ17船団を撃沈した」

正確には、ティルピッツはPQ17船団に一発も砲弾を発射していない。PQ17を撃沈したのはUボートとルフトヴァッフェ（ドイツ空軍）である。ティルピッツの「貢献」は「出撃するかもしれない」という脅威による散開命令の発令まで。

### 誤解②：「ティルピッツは完全に使えなかった失敗作だった」

ティルピッツを「失敗」と評するのは、戦艦の価値を「撃沈した艦数」で測るためである。Fleet in Beingという視点で見れば、ティルピッツは確かに機能した。問題は「そのコストに見合う効果があったか」であり、この問いへの答えは歴史家の間でも分かれる。

### 誤解③：「ティルピッツは一切動かなかった」

実際にはPQ12船団追撃（1942年3月）、スピッツベルゲン攻撃（1943年9月）など、複数の出撃がある。ただしこれらはいずれも本格的な艦隊決戦には至らなかった。

## 現代への教訓

### 教訓①：「存在することの戦略的価値」

現代においても、大型兵器が「使われない」ことで機能している例は多い。

- 核兵器：一発も撃たれていないが、その存在が抑止力として機能している（MAD理論）
- 空母打撃群：実際に戦闘に投入されなくても、派遣されるだけで相手国の行動を制約する
- 弾道ミサイル：移動・展開するだけで相手の意思決定に影響を与える

ティルピッツの事例は、「最強の兵器は、使わないことで最もよく機能する場合がある」という逆説を歴史的に実証した最初の大規模事例の一つである。

### 教訓②：「情報の不確かさが意思決定を歪める」

PQ17事件の本質は「情報の欠如」が「最悪のケースへの対応」を引き出し、より大きな損害を生んだことにある。完全な情報が得られない状況での意思決定がいかに危険かという教訓は、軍事だけでなく企業のリスク管理・危機管理にも直接応用できる。

### 教訓③：「技術革新は既存の前提を覆す」

戦艦は長らく「最強の海軍力」の象徴だった。しかし、航空機・潜水艦・精密爆撃という技術革新が登場したとき、戦艦の前提——「最も重武装した水上艦が制海権を持つ」——は崩れた。ティルピッツを最終的に沈めたのは、戦艦でも空母でもなく、陸上爆撃機だった。

現代でも同様の転換は起きている。ステルス技術・ドローン・サイバー攻撃・ハイパーソニックミサイルは、既存の軍事的前提を覆しつつある。

### 教訓④：「非対称戦の有効性」

Xクラフト（超小型潜水艦）という16mの兵器が、250mを超える超弩級戦艦を半年間戦闘不能にした。トールボーイという「爆弾」が、どんな砲弾も貫通できなかった装甲を突き破った。

大きく・強くあることが必ずしも最善の防御ではない。攻撃者が創意工夫を持って「強さの前提」を外した方法で挑む非対称戦は、現代でも有効であり続けている。

## 歴史的因果関係の整理

```
【ティルピッツをめぐる歴史的因果連鎖】

【第一次世界大戦（1914〜1918年）】
英独海軍軍拡競争 → ユトランド沖海戦（決定的勝利なし）
→ ドイツ海軍の「未完の復讐心」が残る
　↓
【1919年】ヴェルサイユ条約
→ ドイツ海軍の壊滅的制限
→ スカパ・フローでの自沈（「屈辱よりも自沈を選ぶ」という文化的残影）
　↓
【1933〜35年】ヒトラー再軍備・英独海軍協定
→ Z計画の策定
→ ビスマルク・ティルピッツの建造開始
　↓
【1939年9月】第二次世界大戦開戦
→ Z計画中断。不完全な艦隊のまま
→ 「面目のためだけの海軍」とレーダーが嘆く
　↓
【1941年5月】ビスマルク沈没
→ ヒトラー、大型艦に不信感
→ ティルピッツの積極作戦に制限がかかる
→ ティルピッツはノルウェーへ移動し「存在の抑止力」として機能
　↓
【1942年1月〜】ティルピッツのノルウェー配備
→ 北極船団に対する「潜在的脅威」が恒常化
→ イギリス本国艦隊は常時ティルピッツ対応戦力を維持
→ 北極ルートの危険度が急上昇
　↓
【1942年7月4日】PQ17散開命令
→ ダドリー・パウンドがティルピッツの動向に過剰反応
→ 商船24隻喪失・153名死亡
→ イギリスがソ連への責任を果たせず、同盟関係が緊張
→ 北極船団は一時停止（8〜11月）
　↓
【1942年12月】バレンツ海海戦での失敗
→ レーダー解任・デーニッツ就任
→ ドイツ海軍の戦略がUボート一本化に
→ ティルピッツの組織的地位がさらに低下
　↓
【1943年】連続する攻撃と修理の繰り返し
→ Operation Source（超小型潜水艦）→ 6ヶ月戦闘不能
→ Operation Zitronella（スピッツベルゲン攻撃）→ 唯一の実戦砲撃
→ 北岬海戦でシャルンホルスト撃沈→ティルピッツが最後の戦艦に
　↓
【1944年春〜秋】連続する空からの攻撃
→ Operation Tungsten（空母艦載機）→ 沈まず、艦載機の限界露呈
→ Operation Mascot・Goodwood（空母）→ 全て失敗
→ 任務がRAF爆撃機部隊に移管
→ Operation Paravane（トールボーイ）→ 戦闘艦として廃棄
　↓
【1944年11月12日】Operation Catechism
→ トールボーイで沈没
→ 連合国のノルウェー監視の必要が消滅
→ 本国艦隊の資源を他に転用可能に
→ 「ドイツ海軍最後の超弩級戦艦の喪失」＝「戦艦時代の最終的な終焉」を象徴
　↓
【その後への影響】
→ ドイツ海軍の組織的抵抗能力が大幅低下
→ 1945年5月、ドイツ降伏
```
