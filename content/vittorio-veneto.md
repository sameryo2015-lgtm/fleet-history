## 概要

ヴィットリオ・ヴェネト（Vittorio Veneto）は、1930年代のイタリア王国海軍（Regia Marina）が建造したリットリオ級高速戦艦の2番艦である。381mm砲9門・速力30ノット・排水量約4万トンを誇り、その名は第一次世界大戦末期の決定的勝利に由来する。

しかし、この艦の本当の価値は性能そのものにはない。

ヴィットリオ・ヴェネトという一隻の艦を追うことは、

- ムッソリーニが掲げた「地中海はわれらの海（Mare Nostrum）」という帝国主義的野望の実体
- フランス海軍との軍拡競争と、ワシントン条約体制の崩壊過程
- タラント空襲という「艦隊の時代の終わりの始まり」を告げた歴史的事件
- マタパン岬沖海戦での暗号解読（Ultra）と情報戦による敗北
- 燃料・補給・空海統合の欠如が戦艦をいかに無力化するか
- フリート・イン・ビーイング（現存艦隊主義）という消極的戦略の合理性と限界
- イタリア降伏という体制崩壊時の軍隊の忠誠心と「名誉」の問題

……という、第二次世界大戦の地中海戦線を理解するための核心的テーマをすべて横断することである。

## 一言でいうと

> **「地中海制覇の夢を体現しながら、レーダーも空母も燃料も持たずに戦った、強力だが制約された旗艦」**
> 

ヴィットリオ・ヴェネトは、ムッソリーニが「新ローマ帝国」の栄光として建造させた艦であり、実戦では何度も被弾しながら沈まず、タラント空襲では奇跡的に無傷で生き残った。しかし、その軍事的潜在能力は、レーダー技術の欠如・燃料不足・空軍との協同失敗によって最後まで発揮されることがなかった。艦そのものではなく、「なぜそれが活かせなかったのか」という問いにこそ、この艦の歴史的教訓がある。

---

## なぜ有名なのか

### ① タラント空襲での「奇跡的生還」——戦争の転換点の唯一の証人

1940年11月11日夜、英空母HMS イラストリアスから発進したソードフィッシュ雷撃機21機がタラント港を奇襲した。イタリア海軍は史上初の大規模艦載機夜間攻撃を受け、戦艦3隻が中大破・着底した。しかし、外泊地に係留されていたヴィットリオ・ヴェネトだけは魚雷をすべて回避し、無傷で生還した。

この一事が、後の艦の運命を方向づける。翌日、損傷した姉妹艦リットリオの代わりに艦隊旗艦に就任した本艦は、以後イタリア海軍の象徴的存在として英海軍の執拗な追撃を受け続ける。チャーチルが議会で「麻痺的打撃（paralyzing blow）」と形容したこの夜襲は、半年後の真珠湾攻撃の直接のモデルになったとも言われる。

### ② マタパン岬沖海戦——情報戦と技術格差が決めた「見えない敗北」

1941年3月28〜29日のマタパン岬沖海戦では、英ブレッチリー・パークによる暗号解読（Ultra）がカニンガム提督に事前情報を与え、イタリア艦隊の行動が事前に把握されていた。ヴィットリオ・ヴェネトは艦載機の魚雷攻撃で大破し、重巡洋艦ポーラ・ザーラ・フィウメと駆逐艦2隻を失う大敗を喫した。この海戦は「戦力ではなく情報と技術が勝敗を決める」ことを世界に示した。

### ③ ファシズムの象徴——「新ローマ帝国」の看板艦

艦名はイタリア統一後初の大きな国家的勝利（1918年のヴィットリオ・ヴェネトの戦い）に由来し、ファシスト政権が「過去の栄光」を体制強化に利用する「記憶の政治学」の典型例でもある。ムッソリーニが「地中海はわれらの海」と豪語するための政治的道具でもあった。

### ④ 「フリート・イン・ビーイング」の象徴——存在そのものが戦力

ヴィットリオ・ヴェネトは多くの作戦で実際の戦果を挙げなかったが、その「存在」だけで英海軍に多大な抑止的圧力をかけ続けた。英海軍はヴィットリオ・ヴェネトの動向に常に気を配り、護衛力の配備を強いられた。この「戦わずして相手の資源を消耗させる」という戦略は現代の安全保障論にまで通じる概念である。

## 艦名の由来

ヴィットリオ・ヴェネトという艦名は、イタリア北東部ヴェネト州の都市名であり、同時に第一次世界大戦の最後の決定的会戦「ヴィットリオ・ヴェネトの戦い（Battle of Vittorio Veneto, 1918年10月24日〜11月4日）」に由来する。

この戦いでは、イタリア軍がオーストリア＝ハンガリー帝国軍に大攻勢をかけ、わずか2週間足らずでオーストリア軍を瓦解させ、同帝国を降伏に追い込んだ。これはイタリアにとって第一次世界大戦における最大の軍事的勝利であり、数十万人の捕虜を取った「完全な勝利」として語り継がれた。

ファシスト政権は、この「勝利の記憶」を積極的に政治利用した。ムッソリーニは戦勝の象徴としてのヴィットリオ・ヴェネトを自らの「英雄的ナショナリズム」と結びつけ、艦名にその勝利を冠することで、艦そのものをファシズムの正統性を強調する道具とした。

なお、イタリア海軍には第二次世界大戦後も「ヴィットリオ・ヴェネト」の名を冠した艦が就役している（1969年就役の航空巡洋艦）。勝利の記憶がいかに継承されたかを示す一例である。

| 項目 | 内容 |

| 艦種 | リットリオ級高速戦艦（Fast Battleship） |
| 起工 | 1934年10月28日（トリエステ、カンティエーリ・リウニーティ・デッラドリアーティコ造船所） |
| 進水 | 1937年7月25日 |
| 就役 | 1940年4月28日 |
| 排水量 | 標準：約41,000トン（ワシントン条約の35,000トン制限を超過） |
| 主砲 | 381mm/50口径 3連装×3基（計9門） |
| 速力 | 30ノット超 |
| 乗員 | 約1,800〜1,950名 |
| 設計者 | ウンベルト・プリエーゼ将軍 |
| 退役 | 1948年2月 |
| 解体 | 1950〜1954年（ラ・スペツィア） |

リットリオ級は当初ワシントン条約の35,000トン制限内での設計が意図されたが、実際の排水量は約41,000トンに達し、条約制限を超過した。設計にあたっては、速力・装甲・砲力のすべてを高水準で両立させることが求められ、プリエーゼ将軍が独自の水中防御システム（プリエーゼ防御システム）を開発した。ただし、このシステムは実戦での魚雷被弾時に期待された防御力を発揮しなかった。

姉妹艦はリットリオ（Littorio、後にイタリアと改名）・ローマ（Roma）・インペロ（Impero、未完成）の3隻。

## 建造された時代背景

### イタリア王国

イタリアは1861年に統一を達成した比較的新しい国民国家であり、列強の中では後発だった。第一次世界大戦ではイギリス・フランス・ロシアとの協商側に立ち参戦し（「ロンドン条約」1915年）、最終的に勝利国となったが、パリ講和会議での領土問題をめぐって「切り取られた勝利（vittoria mutilata）」と感じ、深い不満を抱えた。

この「勝利したはずなのに報われなかった」という集合的トラウマが、1920年代のファシズム台頭を準備した土壌となった。イタリア社会では、議会制民主主義への幻滅、社会主義勢力との対立、経済的混乱が重なり、強力なリーダーシップを求める声が高まっていた。

```
【イタリア統一（1861年）】
→ 列強の中で最も新参・資源貧弱な「後発帝国主義国」
　↓
【第一次世界大戦（1915〜1918年）参戦・勝利】
→ ロンドン条約で約束された領土が得られず「切り取られた勝利」
→ 国民的不満・民族主義の台頭
　↓
【1919〜1921年：赤い2年間（Biennio Rosso）】
→ 労働争議・社会主義運動の激化
→ 支配層・中間層の恐怖と反動
　↓
【1922年：ローマ進軍とムッソリーニ政権成立】
→ ファシスト政権による「新秩序」の構築へ
```

### ムッソリーニ政権

ベニート・ムッソリーニ（1883〜1945）は、1922年10月の「ローマ進軍」によりイタリア首相に就任した。当初は議会制度の枠内で権力を握ったが、1924年のマッテオッティ事件（社会主義者議員の暗殺）を契機に独裁体制を確立し、一党支配のファシスト国家を作り上げた。

ムッソリーニの外交・軍事政策の核心は「新ローマ帝国」の再興という夢想にあった。ローマ帝国が地中海を「内なる海（Mare Nostrum）」として支配したように、ファシスト・イタリアもまた地中海の覇権を握ることを目指した。そのためには、地中海を「内海」として管理する強力な海軍が不可欠だった。

しかし、ムッソリーニ政権の経済基盤は脆弱だった。イタリアは重要な資源（石炭・石油・鉄鉱石）のほとんどを輸入に依存しており、重工業の規模もイギリス・フランス・ドイツに遠く及ばなかった。アウタルキー（自給自足経済）を掲げながら、実態は外部依存の深い「資源小国のファシズム」だった。

### 地中海戦略

イタリアが置かれた地政学的立場は、一見有利に見えてその実きわめて厳しいものだった。地中海の中央部に突き出したイタリア半島は、東西地中海の連絡線を制御できる位置にあった。しかしムッソリーニの言葉を借りれば、イタリアは「地中海に囚われている（imprisoned in the Mediterranean）」存在でもあった。

```
【地中海の戦略地形】

　ジブラルタル（英）          スエズ運河（英）
　　　‖                           ‖
　　　‖←───地中海──────────────‖
　　　‖                           ‖
　　マルタ島（英）
　　　↑
　　イタリア半島の南方に位置
　　イギリスにとっての「咽喉部」
　　イタリアにとっては「脅威」
```

英国はジブラルタル・マルタ・スエズという「3つの鍵」を握ることで地中海を事実上の内海としていた。イタリア海軍参謀総長カヴァーニャリは開戦前から、「英艦隊がジブラルタルに一つ、スエズにもう一つ展開すれば、われわれは地中海の中で窒息する」と警告していた。この認識は正確であり、実際の戦争でも地中海への「入口」と「出口」を英国が握り続けたことが、イタリア海軍の行動を制約し続けた。

マルタ島はこの地政学的構造の要石だった。英国が保持するマルタは、地中海のほぼ中央に位置し、イタリアのリビア（北アフリカ植民地）への補給線に直接脅威を与えた。イタリアがマルタを早期に制圧できなかったことは、戦略的に致命的な失敗となった。

### フランスとの海軍競争

1920〜30年代の仏伊海軍関係は、複雑な競争と模倣のサイクルだった。その連鎖構造は以下のように整理できる。

```
【仏伊海軍軍拡の連鎖】

1922年：ワシントン海軍条約
→ 仏・伊それぞれに追加70,000トンの新戦艦建造枠を付与
　↓
1929〜32年：フランス、ダンケルク級（26,500トン、330mm砲）起工
→ ドイツのポケット戦艦（ドイッチュラント級）への対抗
　↓
1934年：イタリア、リットリオ・ヴィットリオ・ヴェネト起工
→ ダンケルク級への対抗として、381mm砲9門の高速戦艦を選択
　↓
1935年：フランス、リシュリュー起工
→ リットリオ級への対抗として、380mm砲8門の高速戦艦を選択
　↓
1938年：フランス、ジャン・バール起工
→ 仏伊戦艦競争はワシントン条約の崩壊と共に加速
```

この連鎖は、ダンケルク研究ノートで詳述しているように、本来ドイツのポケット戦艦（グラーフ・シュペー等）への対抗として始まったフランスの戦艦計画が、イタリアを刺激し、さらにフランスを刺激するという「軍拡の螺旋」を生んだ。ヴィットリオ・ヴェネトは、この螺旋の中間点に位置する。

ワシントン条約体制は仏伊に同等の建造枠を与えた（「仏伊パリティ」）が、この「平等」はイタリア側には不満の種だった。イタリアはフランスより強大であるべきだ、という民族主義的自負心が、条約の上限ギリギリ（あるいはそれを超えて）まで艦艇を建造させる動機となった。

## なぜヴィットリオ・ヴェネト級が建造されたのか

### ① フランスのダンケルク級への直接的対抗

1932年、フランス海軍がダンケルク級（ダンケルク・ストラスブール）の起工を発表した際、イタリア海軍にとって衝撃的な事態が生じた。ダンケルク級は330mm砲8門・30ノット超という性能を持ち、当時のイタリア主力戦艦（コンテ・ディ・カヴール級改装型等）では太刀打ちできなかった。

リットリオ級の設計目標はシンプルだった。「ダンケルクをつかまえ、撃破できる艦」——それだけである。381mm砲（フランスの330mmより口径大）・30ノット超（ダンケルクに追いつける速力）という要件が定まると、設計はそれに向けて組み立てられた。

### ② ムッソリーニの政治的要請

純粋な軍事的必要性だけでなく、ムッソリーニの政治的意志が強力に建造を後押しした。「地中海の支配者」にふさわしい看板艦が必要だった。ファシズムの統治には、国民に「われわれは強い」と示す視覚的・象徴的な道具が不可欠だった。巨大戦艦の進水式はそのための格好のメディア・イベントだった。

しかし、この政治的動機は軍事的合理性と必ずしも一致しなかった。海軍参謀長カヴァーニャリは、建造費と運用コストに見合う戦略的価値について終始懐疑的だった。結果として建造された艦は、「使える状況が限られているが使わないわけにもいかない」という矛盾した存在になった。

### ③ ワシントン条約体制の変質

ワシントン条約（1922年）は各国の主力艦の総排水量と新造艦の規模を制限したが、1930年代に入ると日本の脱退（1936年）・ドイツの再軍備（1935年）などにより、条約体制は事実上崩壊しつつあった。

リットリオ級は建造開始時こそ条約枠内での設計を意図していたが、実際の排水量は約41,000トンに達し、35,000トン制限を大幅に超過した。ただし条約自体の「エスカレーター条項」（一国が規定を超えると他国もそれを上限として増やせる）が発動する形で、国際的に묵認された側面もある。いずれにせよ、リットリオ級の誕生はワシントン体制の終焉を象徴していた。

## 艦歴

### 就役〜開戦初期（1940年4〜8月）

ヴィットリオ・ヴェネトは1940年4月28日に就役し、5月にタラントへ回航して第9師団（Divisione）に編入された。イタリアが英仏に宣戦布告した6月10日時点では、本艦はまだ完全な戦闘準備が整っておらず、8月2日に姉妹艦リットリオとともに正式な作戦配備を宣言された。

開戦直後の数ヶ月、ヴィットリオ・ヴェネトは英国のマルタ行き護送船団（オペレーション・ハッツ等）を迎撃すべく複数回出撃したが、いずれも接触できなかった。偵察機の情報が不正確であること、英艦隊の速力や機動性、そして何よりも「主力艦を失ってはならない」という指揮部の慎重方針が、空振りの原因となった。

### タラント空襲後（1940年11月）

1940年11月11日夜のタラント空襲（Operation Judgement）で、外泊地に停泊していたヴィットリオ・ヴェネトは奇跡的に無傷で生還した。翌日、損傷した姉妹艦リットリオに代わって艦隊旗艦に就任し、カンピオーニ提督が座乗した。

同年11月27日、サルデーニャ島南方でスパルティヴェント岬沖海戦（イタリア名：カポ・テウラーダ海戦）に参加。英護衛部隊（空母アーク・ロイヤル・戦艦ラミリーズ・巡洋戦艦レナウン）と対峙し、ヴィットリオ・ヴェネトは主砲19斉射を発砲して英軽巡マンチェスターに軽微な損傷を与えたが、カンピオーニ提督は敵の優勢を過大評価して交戦を打ち切った。

### マタパン岬沖海戦（1941年3月）

1941年3月27〜29日、ヴィットリオ・ヴェネトはイアキノ提督の旗艦として東地中海に出撃した。英艦隊を撃滅してギリシャへの英軍増援を阻止する作戦だったが、英国はブレッチリー・パークの暗号解読によりイタリア艦隊の出撃を事前に把握していた。

3月28日、ヴィットリオ・ヴェネトは英巡洋艦隊を砲撃したが命中弾なし。同日午後、英空母フォーミダブルの艦載機（フェアリー・アルバコア）が魚雷攻撃を実施し、午後3時19分に左舷後部に命中。約4,000トンの浸水と4〜4.5度の傾斜が生じた。緊急注排水により50分後に航行再開し、タラントへ帰投した。この撤退中、救援に向かった重巡ポーラ・ザーラ・フィウメと駆逐艦2隻が英戦艦の夜戦砲撃によって撃沈された。

修理は1941年7月まで続いた。

### 後期活動（1941〜1942年）

マタパン後、ヴィットリオ・ヴェネトは複数のマルタ護送船団迎撃作戦に参加したが、戦果を挙げることなく終わった。1941年12月13日、シチリア海峡を護送作戦中に英潜水艦HMS Urgeの魚雷1本が命中。大浸水が生じたが艦は生き残り、タラントで修理を受けた。

1942年6月、ヴィガラス作戦（英のマルタへの護送作戦）迎撃に出動したが、敵と直接交戦することなく作戦は終了した。1942年中頃以降、深刻な燃料不足によりイタリア主力艦の出撃はほぼ不可能となった。

### 最後の1年（1943年）

1943年6月、ラ・スペツィアへの連合軍爆撃で2発の爆弾が直撃し大破。以後、修理と移動を繰り返した。1943年9月3日のイタリア休戦協定締結を受け、9月9日に艦隊はマルタへ向けて出港した。途中、ドイツ軍がFritz X誘導爆弾（史上初の実戦使用精密誘導兵器）で攻撃。旗艦ローマが撃沈されたが、ヴィットリオ・ヴェネトは無傷でマルタに到着した。

戦後はエジプト（アレクサンドリア→大苦湖）に抑留され、1946年10月にイタリアへ返還。1947年の平和条約でイギリスへの賠償艦として引き渡され、1948年退役・1950〜54年解体。

## タラント空襲

### 背景：地中海の制海権争いの転換点

1940年6月のフランス降伏は、地中海の戦略バランスを劇的に変えた。フランス艦隊という「第二の敵」が消えたことで、英地中海艦隊は単独でイタリア海軍と対峙することになった。同時に、ドイツ軍のギリシャ・エジプトへの進出が英帝国のインド・スエズ路線を脅かした。

英国にとって喫緊の課題はマルタの維持と地中海補給線の確保だった。その最大の障害がタラントに集結するイタリア戦艦部隊だった。

### 作戦計画と実行（Operation Judgement）

英国の艦載機によるタラント攻撃計画は、実はすでに1930年代後半から立案されていた。1940年11月9日の偵察でタラント港にイタリア戦艦6隻すべてが集結していることが確認されると、攻撃の好機と判断された。

1940年11月11日夜（イギリスでは戦勝記念日「ポッピー・デイ」）、HMS イラストリアスからソードフィッシュ雷撃機21機が2波に分けて発艦した。水深約15メートルという浅い港湾での雷撃を可能にするため、魚雷に木製の「転嫁防止翼」を取り付ける改修が行われた。

```
【タラント空襲の結果】
　英攻撃機21機（ソードフィッシュ複葉機）
　　↓
　第1波（12機）：戦艦区に雷撃
　第2波（9機）：戦艦区+補助施設に爆撃
　　↓
　【被弾艦艇】
　・コンテ・ディ・カヴール：魚雷1本命中 → 着底（修理できず終戦）
　・リットリオ：魚雷2本命中 → 着底（修理後復帰）
　・カイオ・ドゥイリオ：魚雷1本命中 → 大破（修理後復帰）
　・ヴィットリオ・ヴェネト：無傷で生還
　　↓
　【英損失】
　ソードフィッシュ2機（乗員4名）
```

### なぜヴィットリオ・ヴェネトは無傷だったのか

ヴィットリオ・ヴェネトは内泊地ではなく外泊地（Mar Grande）に係留されており、魚雷網が設置されていた位置でもなく、かつ英軍の攻撃経路が他の3隻（内泊地のコンテ・ディ・カヴール・リットリオ・カイオ・ドゥイリオ）に集中したことが偶然の幸運だった。少なくとも1本の魚雷が本艦に向けて放たれたが、これは回避された。

「運」と「位置」の組み合わせがこの結果を生んだ。翌朝、損傷した僚艦の代わりに旗艦となったヴィットリオ・ヴェネトは、以後イタリア海軍の「顔」として英軍の執拗な追撃を受けることになる。

### 歴史的影響

タラント空襲は海軍史の転換点となった。

① **艦載機が戦艦を陸上基地なしに攻撃できることの実証**：「港さえ守れば戦艦は安全」という常識が崩れた

② **浅水域での雷撃の可能性の実証**：従来「水深75フィート（22m）以下での魚雷攻撃は不可能」とされていたが、これが否定された

③ **真珠湾攻撃の直接的先例**：日本海軍はタラント空襲を詳細に研究し、1941年12月7日の真珠湾攻撃の作戦立案に応用した（浅水域魚雷・奇襲・夜間攻撃のいずれもが共通している）

④ **地中海の制海権バランスの変化**：英海軍は作戦行動の自由を取り戻し、マルタを拠点としたイタリア補給線への攻撃を強化した

英海軍が21機の旧式複葉機（ソードフィッシュ）で成し遂げたことは、第一次世界大戦のユトランド海戦でドイツ大洋艦隊が受けた損害よりも大きな打撃をイタリア海軍に与えた。

## マタパン岬沖海戦

### 発端：Ultra情報という「見えない切り札」

1941年3月末、英ブレッチリー・パークの女性暗号解析者メイビス・ベイティーが、イタリア海軍のエニグマ変形暗号（C38m）の解読に初めて成功した。彼女が読み解いた情報には、イタリア艦隊（ヴィットリオ・ヴェネトを旗艦とする大部隊）が3月28日に東地中海に出撃する計画が含まれていた。

英地中海艦隊司令長官カニンガム提督はこの情報を受け取った。しかし、暗号解読という手の内を明かさないために、カニンガムは巧みな欺瞞を行った。アレクサンドリアのゴルフクラブに日中わざとらしくスーツケースを持って現れ、「今夜は外泊する」という印象を日本側スパイに与えてから、夜になってこっそり旗艦ウォースパイトに乗り込み出撃した。

### 戦闘の経過（1941年3月27〜29日）

```
【マタパン岬沖海戦の経緯】

3月27日：
・イタリア艦隊（旗艦ヴィットリオ・ヴェネト）がタラント出港
・英艦隊はUltra情報によりすでに出撃準備

3月28日 午前：
・ガウドス島沖で英巡洋艦戦隊（プリダム＝ウィッペル少将）と接触
・ヴィットリオ・ヴェネトが出現、94発発砲するも命中なし
・英巡洋艦隊は後退・煙幕展張

3月28日 午後：
・英空母フォーミダブルのアルバコア雷撃機が攻撃
・15時19分：ヴィットリオ・ヴェネトの左舷後部に魚雷命中
・浸水4,000トン・傾斜4〜4.5度
・損傷した重巡ポーラが行動不能に

3月28日 夜間：
・イアキノ提督、ポーラ救援のためザーラ・フィウメ・駆逐艦を派遣
・英艦隊レーダーにより夜間に接近・奇襲
・22時25〜28分：英戦艦ウォースパイト・ヴァリアント・バーハムが一斉射撃
・ザーラ・フィウメ・ポーラ・駆逐艦2隻が撃沈

3月29日未明：
・ヴィットリオ・ヴェネト、タラントへ帰投（修理に7月まで要した）
```

### イタリア側の敗因分析

マタパン海戦でのイタリアの敗因は、単なる「技術格差」ではなく、複数の要因が重なっていた。

**① 情報の非対称性（最大の要因）**
英国はUltra暗号解読によりイタリアの行動計画を事前把握していた。これに対しイタリア側は英艦隊の動向をまったく把握できていなかった。「見えない情報戦」での完全な敗北だった。

**② レーダーの欠如**
英戦艦HMS ヴァリアントのレーダーは夜間に4,000ヤード（約3.6km）先のイタリア巡洋艦を探知した。イタリア側は夜戦を避けるドクトリンを採用しており、レーダーなしで夜戦に突入した結果、英戦艦の奇襲を受けた。

**③ 艦載機支援の欠如**
英空母フォーミダブルが艦載機で本艦を魚雷攻撃できたのに対し、イタリアは航空母艦を保有していなかった。「イタリア半島そのものが不沈空母だ」というムッソリーニの楽観論が、この致命的弱点を生んだ。

**④ 夜戦能力の差**
英海軍はレーダーによる夜戦射撃を訓練・準備していたが、イタリア海軍は夜戦を回避する方針を採っていた。暗号解読情報を活かすべく、カニンガムは意図的に夜間追撃を選択した。

### カニンガムの「見えない欺瞞」

マタパン海戦の事後、英国はブレッチリー・パークの存在を厳秘にする必要があった。そのため、カニンガムが「どうしてイタリア艦隊の動向を知っていたのか」は戦後長年にわたって謎とされた。1966年にはイタリア海軍武官を通じたスパイ活動説が流布され、実際の1978年の機密解除まで、様々な「偽の説明」が信じられた。このこと自体が、情報戦がいかに戦争の本質を隠蔽するかを示す格好の事例となっている。

## 関連する歴史事件

### ① メルス・エル・ケビール事件（1940年7月3日）

フランス降伏後、英海軍はフランス艦隊がドイツに接収されることを防ぐため、オランの軍港（メルス・エル・ケビール）に停泊するフランス艦隊を攻撃した。1,300名以上のフランス水兵が死亡したこの事件は、仏英間の深刻な亀裂を生んだが、同時に「いまや英海軍が地中海で単独でイタリアと対峙する」という状況を確定させた。この構図がタラント攻撃の決断を加速させた。

### ② マルタ攻防戦（1940〜1942年）

地中海戦争の実質的な中心は、華々しい艦隊決戦ではなく、マルタ島への補給をめぐる地味な攻防だった。英軍はマルタを拠点として枢軸軍のアフリカ補給線を妨害し、イタリア・ドイツは爆撃と機雷で英補給船団を攻撃した。ヴィットリオ・ヴェネトも複数のマルタ関連護送船団迎撃作戦に動員されたが、具体的な戦果を挙げることはなかった。

### ③ 北アフリカ戦線（1940〜1943年）

エジプト・リビアでのロンメル率いるドイツ・イタリア軍と英軍との戦いは、地中海の制海権と直接結びついていた。枢軸軍の兵站はリビア（トリポリ・ベンガジ）への補給に依存しており、英海軍・空軍がこの補給線を断ち切ることを目指した。ヴィットリオ・ヴェネトの作戦行動の多くは、この補給線防衛・護衛と関連していた。

### ④ アレクサンドリア港奇襲（1941年12月19日）

皮肉なことに、ヴィットリオ・ヴェネトのような主力艦が成し遂げられなかった大戦果を挙げたのは、人間魚雷（マイアーレ）を操る特殊部隊だった。6名のダイバーがアレクサンドリア港に侵入し、英戦艦HMS クイーン・エリザベスとHMS ヴァリアントに磁気吸着機雷を取り付けて大破させた。この「人間魚雷」の成功は、巨額の戦艦に匹敵する戦略的打撃を特殊兵器が与えうることを示した。

### ⑤ ロードス・ローマ号の撃沈（1943年9月9日）

1943年9月のイタリア休戦直後、マルタへ向かうイタリア艦隊をドイツ空軍が攻撃した。旗艦ローマ（Roma）にFritz X誘導爆弾（史上初の精密誘導対艦ミサイル）が命中し、約1,200名の乗員と共に沈没した。ヴィットリオ・ヴェネトはこの攻撃をかわして生き延びた。Fritz Xの登場は、「精密誘導兵器が艦艇に対して決定的効果を持つ」時代の始まりを告げた。

### ⑥ 伊降伏と連合国への投降（1943年9月）

1943年9月8日のイタリア休戦協定発表後、イタリア艦隊はドイツに接収されることを避けるためマルタへ向かった。この決断は「ドイツとの同盟への裏切り」か「国王への忠誠と艦隊の名誉の保全」かをめぐって、イタリア国内でも論争が続く問題である。

## 歴史的評価

### 軍事的評価：「強力だが活かされなかった戦艦」

ヴィットリオ・ヴェネトは設計上の性能の面では優秀な戦艦だった。381mm砲9門・30ノット超の速力・充実した装甲は、同時代の英国戦艦（マタパンで戦ったクイーン・エリザベス級はより低速だった）をある面で上回っていた。

しかし、実戦での戦果は限定的だった。多数回の出撃でマルタへの護送船団を阻止できず、タラント空襲・マタパン・潜水艦攻撃・空爆と、被弾・大破を繰り返した。「戦わせてもらえる場面では強かったが、戦える状況が限られていた」という評価が定着している。

軍事史家ヴィンセント・オハラは、ヴィットリオ・ヴェネト級の問題を「艦の欠陥」ではなく「運用上の欠陥」として評価している。設計ではなく使い方の問題だという視点である。

### 戦略的評価：「フリート・イン・ビーイングとしては効果的だった」

ヴィットリオ・ヴェネトが存在し続けたことで、英海軍は常に戦力の一部を地中海に張り付けなければならなかった。英国が「ヴィットリオ・ヴェネトの動向」を常に気にし続けたという事実は、「存在することによる抑止力」が一定の効果を発揮したことを示している。

しかし同時に、この「フリート・イン・ビーイング」戦略は本質的に受動的であり、地中海の制海権を積極的に主張するものではなかった。英国がマルタ補給線を維持し続けた事実は、この戦略が究極的には失敗だったことを示している。

### 技術的評価：「レーダーがなければ戦艦は時代遅れだった」

マタパン岬沖海戦後、イアキノ提督自身が「わが艦隊の敗北は、海空統合の劣位と夜戦技術の遅れを明らかにした」と記した。この分析は正確だった。ヴィットリオ・ヴェネトが2年以上にわたって持ち続けた問題、すなわちレーダー未搭載・空母不保有・空軍との協調不足は、艦そのものの欠陥ではなく、イタリア海軍組織全体の制度的限界を示していた。

### 政治的評価：「ムッソリーニの対外膨張と組織的合理性の衝突」

ムッソリーニは「常に攻勢的であれ」と海軍に命じたが、カヴァーニャリからイアキノに至る海軍指導部は一貫して慎重な「艦隊温存」路線を採った。この対立は、政治的意志と軍事的現実の乖離という、ファシズム体制が陥りやすい構造的矛盾を体現していた。

## 論争点・誤解されやすい点

### 論争①：フリート・イン・ビーイング戦略は「臆病」か「合理的判断」か

タラント空襲後の艦隊温存策は、長年「指導部の臆病さ」として批判されてきた。しかし一方で、「失えば補充できない主力艦を無謀な決戦で失わないことが、英海軍の作戦計画を複雑にし続けた」という再評価もある。カヴァーニャリが危惧した通り、「一度艦隊を失えば地中海に対する影響力がゼロになる」という構造的制約の中での選択だった。

| 立場 | 主張 | 根拠 |
| --- | --- | --- |
| 批判的評価 | 積極的攻勢に出れば地中海の制海権を取れた | タラント直後は英主力艦が3隻大破し、英艦隊が弱体化していた |
| 肯定的評価 | 艦隊温存は合理的だった | 燃料・補給の制約下で消耗戦を避けることは現実的選択。英海軍は常に警戒し続けた |
| 現代的評価 | 「どちらでもなかった」 | 問題は攻撃的か守備的かではなく、空海統合の欠如という構造的問題にあった |

### 論争②：ムッソリーニの「不沈空母発言」は本当にあったのか

「イタリア半島そのものが不沈空母だ」という発言がムッソリーニによるものとして広く引用されるが、この発言の一次史料は明確ではなく、後世の誇張や誤伝の可能性を指摘する研究者もいる。ただし、発言の真偽にかかわらず、イタリアが航空母艦を最後まで持たなかったことは事実であり、その戦略的失敗も論争の余地がない。

### 論争③：マタパン岬沖海戦での敗北はUltraが原因か、戦術的ミスが原因か

戦後長年にわたり、マタパン海戦の真の敗因は「暗号解読（Ultra）」であることが公表されなかった。その間、「イタリア側の戦術ミス（ポーラの救援に向かったこと）」が敗因とされることが多かった。1978年の機密解除後、Ultraの存在が明らかになったが、「Ultraがなくてもカニンガムは勝てたか」という問いは今も続く。

| 見解 | 評価 |
| --- | --- |
| Ultra決定論 | Ultraなしには英艦隊はイタリア艦隊の出撃を事前に把握できず、夜戦での奇襲は不可能だった |
| 戦術ミス論 | ポーラ救援のための艦隊分離という判断ミスが敗因。Ultra情報がなくとも英艦隊は有利だった |
| 統合的見解 | Ultraが状況を決定的に有利にし、レーダー優位が夜戦での圧倒的勝利を可能にした。いずれか一つの要因だけでは説明できない |

### 論争④：ベルガミーニ提督は自沈を望んでいたのか

1943年9月のマルタ回航の際、ベルガミーニ提督が「マルタではなく自沈を望んでいた」という証言と「提督は命令に従いマルタへ向かった」という証言が対立する。提督がFritz X攻撃で戦死したため、真相は永遠に不明のままである。これは「体制崩壊時の軍人の忠誠心はどこへ向かうか」という普遍的問いを内包している。

### 誤解①：「イタリア海軍は弱かった」という誤解

ヴィットリオ・ヴェネトのような高性能艦を持ちながら戦果を上げられなかったことから、「イタリア海軍は弱かった」という印象が広まった。しかし実際には、潜水艦・人間魚雷・機雷等では顕著な戦果を挙げており（アレクサンドリア港奇襲でQEとヴァリアントを大破）、水上戦艦部隊の不振はイタリア海軍全体の評価に当てはまらない。

### 誤解②：「タラント空襲がすべてを決めた」という誤解

タラント空襲の戦略的影響は大きかったが、「これでイタリアは地中海の制海権を失った」という解釈は過剰である。損傷した戦艦2隻は修理後復帰し、イタリア海軍は1942年中頃まで中央地中海で一定の優位を保った。マルタ行き英護送船団が壊滅的損害を受けたのはタラント後のことである。

## 現代への教訓

### ① 技術的優位は、組織的統合なしには無力である

ヴィットリオ・ヴェネトは個艦として高性能だったが、レーダー未搭載・航空支援の欠如・空軍との協調不足という「システム統合の失敗」により、その性能を十分に活かすことができなかった。現代の軍事力評価においても、個別兵器の性能よりも「システムとしての統合度」が問われる。F-35の価値がネットワーク統合にあるように、「単体の性能」と「システムとしての効果」は別物である。

### ② 情報優位は物理的戦力を凌駕しうる

マタパン岬沖海戦では、英国の暗号解読（Ultra）という情報的優位が、戦力的には互角以上だったイタリア艦隊を決定的に打ち負かした。現代のサイバー戦・情報戦においても、「相手の意図と行動を事前に知っていること」のアドバンテージは決定的である。情報保全の失敗は、いかなる物理的優位をも無効化する。

### ③ 兵站・補給なき作戦力は幻想である

イタリア海軍の最大の制約は、自国で産出できない石油に依存する構造だった。燃料不足により1942年後半以降、主力艦はほぼ出撃不可能になった。「どれだけ優秀な兵器を持っていても、補給線を断たれれば戦えない」という兵站の法則は、古代ローマから現代のウクライナまで変わらない。

### ④ 「存在による抑止力」の有効性と限界

ヴィットリオ・ヴェネトはフリート・イン・ビーイング戦略により、実際に戦わずして英海軍に多大な計画負荷をかけ続けた。しかし究極的には、「存在するだけ」では地中海の制海権を獲得できず、英補給線の遮断も達成できなかった。抑止力は存在するだけでは目標を達成できない——行使できる能力の裏付けが必要であるということを示している。

### ⑤ 政治的野心と軍事的現実の乖離がもたらす悲劇

ムッソリーニの「地中海制覇」という政治的野心は、燃料・レーダー・航空支援・補給能力という軍事的現実によって裏付けられていなかった。「我々の海」を唱えながら、実際には英軍に輸送線を維持され続けた。政治リーダーが軍事力の実態と乖離した目標を設定するとき、現場の将兵が代償を払うのはどの時代も変わらない。

## この艦から広がる歴史

### 地中海戦争というより大きな物語

ヴィットリオ・ヴェネトを起点にして、地中海戦争（1940〜1943年）全体を俯瞰することができる。この戦いは単なる「英伊の海戦」ではなく、ファシズムの対外膨張・ドイツとの不釣り合いな同盟・北アフリカ戦線の命運・真珠湾攻撃への影響・Fritz X誘導爆弾という「最初の精密誘導兵器」の登場といった、第二次世界大戦全体に関わるテーマを内包している。

### タラント空襲から真珠湾攻撃へ

1940年11月のタラント空襲は、1941年12月の真珠湾攻撃の直接の先例となった。日本海軍はタラントを詳細に研究し、浅水域での雷撃技術・夜間奇襲・航空母艦からの大規模攻撃という三要素を真珠湾作戦に応用した。ヴィットリオ・ヴェネトが生き残った一夜が、数千マイル離れた太平洋の歴史を変えた連鎖の起点になった。

### Fritz Xという「最初のミサイル」

1943年9月9日、旗艦ローマを沈没させたドイツのFritz X誘導爆弾は、歴史上初めて実戦で成功した精密誘導兵器とみなされている。遠隔操作による誘導が可能なこの兵器は、現代の対艦ミサイルの先駆けであり、「誘導兵器が戦艦を時代遅れにする」という趨勢を決定的にした。ヴィットリオ・ヴェネトはこの攻撃を回避したが、旗艦ローマの沈没は「誘導兵器の時代の到来」を告げた。

### イタリアの戦後と「きれいな海軍」神話

戦後イタリアでは、「海軍は勇敢に戦ったが、無能な政治家とドイツの身勝手さと資源不足に裏切られた」という自己弁護的な歴史観が長く主流だった（ブラガディン史観）。しかし現代の研究では、海軍指導部自身の官僚主義的限界・空軍との協調失敗・夜戦回避ドクトリンの硬直性といった「内部的失敗」が再評価されている。この「神話の解体」は、ドイツの「きれいな国防軍」神話の解体と並行して進んだ。

### 「海軍軍拡の連鎖」から見た現代

ダンケルク→リットリオ→リシュリューという仏伊海軍軍拡の連鎖は、「一国の軍拡が他国を刺激する」という古典的な安全保障のジレンマの実例である。現代における中国の海軍増強→日本・オーストラリアの対応→アメリカの関与強化という連鎖は、構造的に類似した現象と分析できる。

| **空軍協調** | 海軍と空軍の深刻なセクショナリズム | 艦隊航空隊（FAA）が海軍直属 | **英圧倒的優位** |
| **補給線長さ** | 地中海内（比較的短い） | ジブラルタル〜スエズ（長距離） | 伊やや優位 |
| **特殊部隊** | 人間魚雷（マイアーレ）が卓越した成果 | 特殊潜航艇等 | 伊優位 |
| **潜水艦** | 地中海最多（110隻超） | 地中海配備は少数 | 伊優位 |

**総評：** 個別の艦艇性能では伊が英に勝る部分も多かった。しかし、航空母艦・レーダー・暗号解読・夜戦能力・空海統合という「システム全体の統合度」では英国が圧倒的に優位であり、戦争の帰趨を決したのはこの「システム格差」だった。

## 地中海戦略解説（深掘り）

### マハン理論と地中海

アルフレッド・セイヤー・マハン（1840〜1914）の「海上権力論（Sea Power Theory）」は、「海上支配こそが国家の繁栄と覇権を決定する」と主張した。地中海はその理論が最も典型的に当てはまる「閉じた海」だった。

ヴィットリオ・ヴェネト級の建造は、ムッソリーニがマハン的な「艦隊による制海権の確立」を目指したことを示している。しかし実際には、マハンが重視した「艦隊決戦による制海権の確立」は実現せず、その代わりに「補給線の争奪」「消耗戦」「情報戦」という現代的な戦いの形が地中海を支配した。

### 地中海の3つの「鍵」

英国が地中海を制御できた最大の理由は、以下の3点を押さえていたことにある：

```
【英国の地中海支配の3点】

ジブラルタル（西の扉）
　・アフリカとヨーロッパの間
　・大西洋から地中海への入口を制御
　・H艦隊の基地

マルタ島（中央の要石）
　・地中海ほぼ中央
　・伊・リビア間の補給線を脅かす位置
　・英潜水艦・航空機の基地
　・最終的には生き延び、北アフリカ反攻の拠点に

スエズ運河（東の扉）
　・アジア・インドへの最短ルート
　・地中海から紅海への出口を制御
　・英帝国の命脈
```

イタリアはこのいずれも保有せず、地中海の「入口」と「出口」と「中央の要石」をすべて英国に握られたまま戦い続けた。

### フリート・イン・ビーイング（現存艦隊主義）の論理

フリート・イン・ビーイング（Fleet in Being）という戦略概念は、「艦隊が実際に戦わなくても、存在するだけで相手の計画に制約を与える」という考え方である。

イタリア海軍がこの戦略を採用した理由は明確だった：

① 艦を失えば補充できない（英国に比べ工業力・資金力が劣る）
② 主力艦の喪失は「戦争継続能力の喪失」を意味する
③ 英国がマルタ・ジブラルタル・スエズを押さえている限り、艦隊決戦で勝っても制海権を取れない

しかしこの戦略には根本的な問題があった：

**「存在による抑止力」は、「行使できる脅威の信憑性」に依存する。**

タラントで3隻が大破した後、英国はイタリア艦隊が積極的には出てこないと判断し、マルタへの補給作戦を継続した。「存在」だけでは、英国のマルタ維持という根本的な戦略目標を妨げられなかった。

## 歴史的因果関係の整理

ヴィットリオ・ヴェネトの物語を、単発の出来事の集まりではなく、一本の因果の鎖として整理する。

```
【第一次世界大戦（1914〜1918年）】
伊の「切り取られた勝利（vittoria mutilata）」
→ 民族主義的不満の蓄積
　↓
【1919〜1922年】
経済混乱・社会主義運動・議会制への幻滅
→ ファシスト運動の台頭
　↓
【1922年】ムッソリーニ政権成立
→「新ローマ帝国」・「地中海はわれらの海（Mare Nostrum）」路線
　↓
【1922年】ワシントン条約（海軍軍縮）
→ 仏・伊それぞれに70,000トンの建造枠
→「仏伊パリティ」という制約がイタリアを刺激
　↓
【1929〜32年】フランス、ダンケルク級起工
→ ドイツのポケット戦艦への対抗
→ イタリアが「追いつかなければならない」と判断
　↓
【1934年】リットリオ・ヴィットリオ・ヴェネト起工
→ ダンケルク級への対抗として381mm砲高速戦艦を選択
→ 35,000トン条約制限を実質的に超過
　↓
【1935年】フランス、リシュリュー起工
→ リットリオ級への対抗（軍拡の螺旋が加速）
　↓
【1935〜36年】エチオピア侵攻→英仏との対立激化
→ 枢軸同盟（伊独）へのシフト
→「英仏が主敵」という前提が確定
　↓
【1940年6月10日】イタリア参戦
→ ヴィットリオ・ヴェネトが地中海作戦に投入
→ しかし燃料・偵察・空海統合の欠如により空振り続出
　↓
【1940年11月11〜12日】タラント空襲
→ 英艦載機21機がイタリア戦艦3隻を大破
→ ヴィットリオ・ヴェネトのみ無傷・旗艦就任
→ 「港も安全ではない」という航空時代の始まりを告げる
→ 日本海軍がこれを研究→真珠湾攻撃の雛型に（1941年12月）
　↓
【1941年3月】マタパン岬沖海戦
→ Ultra暗号解読により英艦隊がイタリアの動向を把握
→ ヴィットリオ・ヴェネト、魚雷被弾・退避
→ 重巡3隻・駆逐艦2隻喪失（伊海軍史上最大の敗北）
→「情報戦の優位が物理的戦力格差を覆す」ことが証明された
　↓
【1941〜1942年】燃料不足の深刻化
→ ドイツ依存の補給体制が限界
→ イタリア主力艦の出撃ほぼ不可能に
→ 「存在による抑止力」も意味を失う
　↓
【1942年後半〜1943年】連合軍の北アフリカ反攻
→ エル・アラメインの戦い（1942年10月）でロンメル敗北
→ シチリア上陸（1943年7月）→本土への脅威
→ イタリア王政府がムッソリーニを解任（1943年7月25日）
　↓
【1943年9月】イタリア休戦協定
→ ヴィットリオ・ヴェネト、マルタへ回航
→ 旗艦ローマ、Fritz X（史上初の精密誘導爆弾）で撃沈
→ ヴィットリオ・ヴェネト、またしても奇跡的生還
　↓
【戦後】
賠償艦として英国に引き渡し・解体（1948〜1954年）
→「ファシズムの夢」の物質的終焉
　↓
【現代への教訓】
・情報優位の決定的価値（Ultra→現代のサイバー・SIGINT）
・システム統合なき個別性能の限界
・兵站・補給の戦略的重要性
・政治的野心と軍事的現実の乖離が招く悲劇
```

> **【ヴィットリオ・ヴェネトという入口の強み】**
> 
> 
> ダンケルクが「ポケット戦艦への対抗から生まれた革新」への入口であり、グラーフ・シュペーが「情報戦・心理戦による砲弾なき敗北」への入口であったように、ヴィットリオ・ヴェネトは**「技術的に優秀な艦が、システム統合の失敗・情報劣位・補給の欠如によって潜在能力を発揮できなかった」という構造的失敗**への入口として機能する。
> 
> 一隻の艦の物語から出発し、**ファシズムの政治的野心 → 仏伊海軍軍拡競争 → ワシントン条約体制の崩壊 → タラント空襲という戦争の転換点 → Ultraという情報戦の奇跡 → フリート・イン・ビーイングの合理性と限界 → Fritz Xという精密誘導兵器の誕生 → そして現代のサイバー戦・精密ミサイル・情報セキュリティへ**と、自然に広がることができる。
> 
> 本シリーズの他の研究ノート――ダンケルク（本艦のリットリオ級を生んだ対抗艦）、リシュリュー（次世代の対抗艦）、プリンス・オブ・ウェールズ（航空機が戦艦を沈めることを確定させたマレー沖海戦）、ティルピッツ（同じ「フリート・イン・ビーイング」に縛られたドイツ側の事例）、グラーフ・シュペー（情報戦・心理戦の先例）――と合わせて読むことで、この艦が持つ歴史的な広がりはより立体的に理解できる。
